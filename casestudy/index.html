IDM240 - Interactive Graphics

Dylan Roussin

Summary

For this project, the class was tasked to create an interactive poster for a musical artist, leveraging scaleable vector graphics (SVG) as a means for animation and interactivity.

Challenge

The largest challenge in this project was using the SVG paths as the catalyst for animation. Javascript was required to select parts of the SVG which could then be used to alter aspects like color and position of the indiviudal elements within an SVG. Requirements for this project included a song that could be paused and played, as well as a looping and non-looping animation. I choose the artist Phish for this project, and specifically their event called "The Baker's Dozen." The theme uses donuts heavily, which was used heavily in the project.

Solution

A number of SVGs were implemented including:

  • Amplifier logo: changes color based on the music's state. Pushes in and out when the user clicks.
  • "The Baker's Dozen" logo: spins 360 degrees when the user clicks.
  • Falling donuts: donuts fall at staggered intervals in what appears as tubes in a looping animation.
  • Conveyer belt: another donut repeatedly slides down a conveyer belt.
  • Donut bounce: the sprinkles and frosting of the brown donut pop up when the user clicks.

I used the song "Simple" by Phish was used for this project. Overall the poster's colors and factory feel acomplish the goals I had for this project, and effectively communicate the identity of the event.
